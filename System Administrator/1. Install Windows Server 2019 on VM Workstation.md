# Install Windows Server on a Virtual Machine

This guide shows how to install **Windows Server 2019** on a Virtual Machine (VMware Workstation). The steps are similar for other Windows Server versions and different hypervisors.

## 1. Download Windows Server ISO

Download the ISO from the [Microsoft Evaluation Center](https://www.microsoft.com/en-us/evalcenter/):


[- Windows Server 2025]([https://example.com](https://www.microsoft.com/en-us/evalcenter/download-windows-server-2025))
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2

<img width="885" height="555" alt="image" src="https://github.com/user-attachments/assets/6324ce71-4f2d-496b-b816-fdbba30ff509" />

Choose the **64-bit edition** in your preferred language.

## 2. Mount the ISO

1. Create a new Virtual Machine.
2. Edit VM settings and attach the Windows Server ISO to the CD/DVD drive.
3. Save and power on the VM.

## 3. Install Windows Server
<img width="683" height="525" alt="7" src="https://github.com/user-attachments/assets/6011a3dc-6900-423b-8023-a68c309863d3" />

1. Press any key to boot from the ISO when prompted.
<img width="781" height="450" alt="image" src="https://github.com/user-attachments/assets/ef26d4ef-02de-4858-bb57-38052572fffa" />


2. Select language preferences → **Next**.
<img width="1090" height="817" alt="image" src="https://github.com/user-attachments/assets/4f3eb22d-481f-401d-8ceb-1ec7e116ac3b" />

4. Click **Install now**.

5. Select **Windows Server 2019 Standard Evaluation (Desktop Experience)** → **Next**.
<img width="683" height="525" alt="7" src="https://github.com/user-attachments/assets/4fc1dbf7-90e2-44d4-a720-14103247673c" />

7. Accept the license terms → **Next**.
<img width="1182" height="674" alt="2026-03-25 08_53_22-Windows Server 2019 - VMware Workstation" src="https://github.com/user-attachments/assets/5c3d2089-3a09-47fa-a171-59f605bb5e99" />

8. Choose **Custom: Install Windows only (advanced)**.
<img width="1090" height="817" alt="image" src="https://github.com/user-attachments/assets/d4294133-6847-4cc3-9d9e-f598d6d2de05" />

9. Select the drive with **Unallocated Space** → **Next**.

The installation will proceed through:
- Copying files
- Installing features and updates
- Completing setup

<img width="1182" height="674" alt="2026-03-25 08_58_29-Windows Server 2019 - VMware Workstation" src="https://github.com/user-attachments/assets/abca7662-1156-4315-9d88-504a7516272f" />

The VM will restart several times.
<img width="1182" height="674" alt="2026-03-25 08_59_33-Windows Server 2019 - VMware Workstation" src="https://github.com/user-attachments/assets/8db001e2-9e89-4e3a-b37b-f241630bda6c" />

8. Set a password for the built-in **Administrator** account → **Finish**.
<img width="1002" height="705" alt="2026-03-25 09_03_03-Windows Server 2019 - VMware Workstation" src="https://github.com/user-attachments/assets/883c1fa7-5cce-45b4-a274-7e7ee5dde92e" />

## 4. Sign In

1. Press **Ctrl + Alt + Delete** at the lock screen.
2. Enter the Administrator password.
<img width="951" height="732" alt="2026-03-25 09_04_40-Windows Server 2019 - VMware Workstation" src="https://github.com/user-attachments/assets/7102a266-fb5c-4e83-9368-bdcdfb192eba" />

3. Server Manager opens automatically.
<img width="1034" height="768" alt="2026-03-25 09_09_25-Windows Server 2019 - VMware Workstation" src="https://github.com/user-attachments/assets/625e5da0-3e31-44da-82de-61822f5d61c9" />

**Windows Server is now successfully installed.**
